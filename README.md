# Projeto-Podcast-com-IAs
Este projeto foi desenvolvido como parte do desafio final do Bootcamp Prompts Inteligentes - formação em Engenharia de Prompts para IAs. O objetivo é aplicar, de forma criativa e prática, os conhecimentos adquiridos durante a formação, utilizando inteligências artificiais generativas para conceber, planejar, automatizar e produzir um Podcast
